# 背景
- 为了适应两个的多人2v2环境，需要禁止玩家在地图的随机事件中死亡
- 采取的手段是当事件涉及到掉血时，把掉血量的上限修改为当前HP-1
- 由于每个事件的掉血逻辑都是单独实现的，因此需要对这些选项做分类讨论。
